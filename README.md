# Trivia Visualizer

The React + Vite web app that visualizes data from [Open Trivia DB](https://opentdb.com/). It provides distribution charts for categories and difficulty leveles and features interactive category filtering.

## Installation

Clone the repository
```
git clone https://github.com/arinash/trivia-visualizer.git
cd trivia-visualizer
```
Install dependencies ```npm install```

Run ```npm start```

Runs the app in the development mode.

Open URL shown in your terminal to see it in your browser (usually http://localhost:5173).

## Live Demo
https://arinash.github.io/trivia-visualizer/

## API Usage

Questions are fetched from https://opentdb.com/api.php

Categories are fetched from https://opentdb.com/api_category.php
